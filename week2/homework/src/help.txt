Usage: node index.js [options]
FooCoding Node.js Week 2 - command line to do
Options:

  list          reads all to-dos list

  add [to-do]   adds to list (eg. add laundry)

  remove        takes number as argument of item to remove (eg. remove 2) removes second task

  reset         clears all list items

  update        takes number to update as first argument and item to update with as second argument
                (eg. update 4 laundry) replaces task 4 with laundry

  help          show this help text
